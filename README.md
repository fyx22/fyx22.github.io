# xiuxian
修仙之路  - 小游戏 玩法同2048

<a href="http://www.uexts.com/index.php/Games/xiuxian" target="_blank">线上地址</a>

浏览器支持:现代浏览器

在2048的逻辑基础上增加修仙等级对应的文字映射,以当前游戏程序为基础,开发者可以开发出任意等级,只需要修改 `game_manager.js` 中的`this.tilesTextArr`对应的文字以及`tile.js`中的`this.text `默认初始等级即可.

`修仙之路`对应的等级有 '旋照','开光','融合','心动','灵寂','元婴','出窍','分神','合体','渡劫','大乘'共11个等级,与`2048 = 2^11`对应,若不需要这么多等级或者有更多的等级需求,除了修改上述两个地方外,还需要修改2048为对应的2的次方数.
